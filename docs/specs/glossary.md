# 用語集

## body

キーキャップ本体の主要体積。印字とは別に扱えるように設計する候補。

## legend

キーキャップ上の印字や記号を表す形状。色属性だけではなく、別体積として出力できる構造を優先候補とする。

## separate volume / 別体積方式

body と legend を別ボディとして保持し、出力形式やスライサー側で独立に扱えるようにする設計方針。

## preview

ブラウザ上で編集結果を即時確認するための表示経路。軽量化や簡略化を許容する。

## export

最終的な STL / 3MF などの出力経路。寸法と形状の正確性を優先する。

## profile

キーキャップの高さ、傾き、上面形状など、全体形状の系統を表す概念。

## preset

UI 初期値や SCAD パラメータの組み合わせをまとめた定義。キー種別や profile 差分の管理に使う。

## stem

キーキャップ裏面の取り付け形状。方式差分が大きいため、独立モジュール化しやすい。

## PoC

Proof of Concept。正式採用前に成立可能性を確認するための検証。
