# Font Source Note Templates

## OFL Font

```text
<Font Name> source and license note
===================================

Source page:
<official source page>

Source page at review:
<commit/tag fixed source page when available>

Bundled download URL:
<commit/tag fixed raw font URL>

Bundled license URL:
<commit/tag fixed raw OFL URL>

Download date:
YYYY-MM-DD

Review date:
YYYY-MM-DD

Bundled files in this repository:
- <FontFile>.ttf
- <Font>-OFL.txt

Bundled SHA-256:
- <FontFile>.ttf: <sha256>
- <Font>-OFL.txt: <sha256>

License evidence:
- <official source or OFL text evidence>
- The bundled TTF nameID 13 says: <license description>

Font metadata observed from the bundled TTF:
- nameID 0: <copyright>
- nameID 1: <family>
- nameID 4: <full name>
- nameID 5: <version>
- nameID 13: <license description>
- nameID 14: <license URL>

Use note:
The bundled font is used unmodified as a legend font asset and redistributed with its OFL text.
```
